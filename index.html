Objetivo

Probar y documentar ejemplos mínimos de integración (callbacks, mapping de campos, firmas HMAC) para handoffs con adquirentes/partners.

Endpoints de ejemplo (mock)

  • Autorización: POST /v1/payments/authorize
  • Notificaciones: POST /v1/webhooks/settlement

Nota: Este sitio es solo demostrativo; no procesa datos reales.

Checklist de handoff

  1. Contrato de API y ejemplos (request/response) adjuntos.
  2. Esquema de firma (HMAC/keys) y rotación.
  3. Ambientes: Sandbox → UAT → Prod.
  4. Plan de pruebas: casos felices, borde y error.
  5. Playbook de rollback y contact list 24/7.
